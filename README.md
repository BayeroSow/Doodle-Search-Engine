# Search Engine
This is a search engine that analyzes a content of a page, catalogs images and video files embedded on the page.


# Requirements
- Crawl website
- Search for links and images
- Pagination system


# Tech Stack
- Visual Studio Code
- Ajax
- Xamp
- Third party tools
